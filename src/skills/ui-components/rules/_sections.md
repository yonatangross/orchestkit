# Rule Categories

## 1. shadcn/ui (shadcn) — HIGH — 3 rules

Beautifully designed, accessible component patterns with CVA variants, cn() utility, and OKLCH theming.

- `shadcn-customization.md` — CVA variant system, cn() class merging, OKLCH theming, component extension
- `shadcn-forms.md` — Form field wrappers, react-hook-form integration, validation error states
- `shadcn-data-table.md` — TanStack Table integration, column definitions, sorting, filtering, pagination

## 2. Radix Primitives (radix) — HIGH — 3 rules

Unstyled, accessible React primitives for building high-quality design systems.

- `radix-dialog.md` — Dialog, AlertDialog, controlled state, Portal, animations, focus trapping
- `radix-composition.md` — asChild polymorphic rendering, Slot, nested triggers, ref forwarding
- `radix-styling.md` — Data attributes, Tailwind arbitrary variants, focus management, keyboard navigation
