# Rule Categories

## 1. Issue Tracking Automation (tracking) — HIGH — 1 rule

Automated progress updates from commits, branch detection, and session summaries. Eliminates manual status updates.

- `issue-tracking-automation.md` — Auto-progress from commits, sub-task completion, session summaries

## 2. Issue-Branch Linking (linking) — MEDIUM — 1 rule

Connecting issues to branches, PRs, and commits for traceability. Enables automatic issue closure and progress visibility.

- `issue-branch-linking.md` — Branch naming conventions, commit references, PR linking patterns
