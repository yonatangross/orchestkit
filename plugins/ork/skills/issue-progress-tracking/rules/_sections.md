# Rule Categories

## 1. Start Work (start) — HIGH — 1 rule

- `start-work-ceremony.md` — Branch from default, label issue, comment intent

## 2. Small Commits (commits) — HIGH — 1 rule

- `small-commits.md` — Atomic commits, issue references, commit frequency
