# Rule Categories

## 1. Dashboard (dashboard) — HIGH — 3 rules

Responsive grid layouts, widget composition, and data table patterns for admin panels and analytics dashboards.

- `dashboard-layout.md` — CSS Grid responsive layouts, sidebar + main area, breakpoint patterns
- `dashboard-widgets.md` — StatCard, widget registry, TanStack Query + SSE real-time updates
- `dashboard-filters.md` — TanStack Table with sorting, filtering, pagination, skeleton loading

## 2. Charts (charts) — HIGH — 3 rules

Recharts 3.x data visualization with responsive containers, custom tooltips, and performance patterns.

- `charts-line-bar.md` — LineChart, BarChart, AreaChart, PieChart with proper configuration
- `charts-responsive.md` — ResponsiveContainer, accessibility with figure/aria-label, parent dimensions
- `charts-custom.md` — Custom tooltips, real-time updates, performance optimization, sliding window
