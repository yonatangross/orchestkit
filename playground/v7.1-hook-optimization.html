CC fires PreToolUse, PermissionRequest, PostToolUse as separate events — can't merge at plugin level. v7.1 optimizes what happens inside each spawn, not the spawn count.

#921 — Cache Project Hash

src/hooks/bin/run-hook.mjs
let cachedPid = null;
function getProjectHash(projectDir) {
  if (cachedPid === null) {
    cachedPid = createHash('sha256')
      .update(projectDir).digest('hex').slice(0, 12);
  }
  return cachedPid;
}
SHA-256 computed once per process. Every subsequent call returns cached value.
