hook-contract 0.1.1 🥇

First release cut through the OIDC pipeline — and the one that reconciles npm ⇔ PyPI to content + version parity.

What 0.1.1 does

  • Proves the OIDC npm pipeline (#2096) end-to-end on a real tagged release — no NPM_TOKEN.
  • Reconciles the registries: npm 0.1.0 (today's bootstrap) shipped signing; PyPI 0.1.0 (2026-05-18) predated it. 0.1.1 brings PyPI's signing module out of "Unreleased" so both registries are content-equivalent at 0.1.1.
  • No contract/code changes beyond release infra + the repository.url normalization. The 19 events + schemas are unchanged.

The FOUR version sources (all must equal the tag)
