chain-state-watcher new

Alerts when .claude/chain/state.json becomes stale (>30min) during multi-phase skills like brainstorm, implement, cover, fix-issue. Suggests /recap or resume.

Trigger: Auto-arms at session start. Checks every 5 minutes.

uncommitted-work-reminder new

Periodic reminder when >10 files have uncommitted changes, suggesting /ork:commit to save progress. Prevents work loss from rate limits or session crashes.

Trigger: First check after 10 min, then every 15 min.

Schema Format

CC 2.1.105 monitors use a top-level JSON array at monitors/monitors.json. Each entry has name, description, command. Auto-arms when plugin loads.

ENABLE_PROMPT_CACHING_1H 2.1.108

Extends prompt cache TTL from 5 minutes to 1 hour. Major cost savings for long sessions (brainstorm, implement, cover chains). Works on API key, Bedrock, Vertex, Foundry.

export ENABLE_PROMPT_CACHING_1H=1
