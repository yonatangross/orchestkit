# Notification Hooks

**3 hooks** registered for the `Notification` lifecycle event.

| Hook | Matcher | Behavior | Description |
|------|---------|----------|-------------|
| `desktop` | `*` | 🔇 Silent | Sends rich desktop notifications with context (repo, branch, task). |
| `sound` | `*` | 🔇 Silent | Plays sounds for task completion using detached child processes |
| `webhook-forwarder` | `*` | 🔇 Silent | Universal Webhook Forwarder — streams ALL hook events to registered sinks. |
