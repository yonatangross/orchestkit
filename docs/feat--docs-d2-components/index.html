How we adopt every Claude Code release

Claude Code ships several releases a week. OrchestKit silently breaks if it falls behind, so the repo runs an automated adoption loop: detect, extract, verify, file, adopt, subtract.

In one July 2026 week, the loop turned three releases into 37 auto-filed, scored adoption issues. Zero were written by hand.
