🚇 What actually ships

Why it wins
  • Zero scroll-JS in production, CSS animation-timeline, compositor-only
  • Signage & departure boards render from generated-data: art can never lie about counts
  • View Transitions morph station glyphs into /docs headers, home and docs feel continuous
  • Degrades by construction: Firefox / reduced-motion / no-JS get the finished readable map
Honest risks
  • Spectacle ceiling, needs the diorama station arrivals + hero scene to feel BIG
  • animation-timeline gaps mean a minority sees the static map
  • Metaphor is load-bearing: every station keeps its literal docs label
Ship slices
  • S1: map spine, pure SVG/CSS, mergeable alone
  • S2: Higgsfield 2.5D station dioramas (lazy, poster-first)
  • S3: Genesis-style hero flight landing into Origin
  • S4: Session Replay chapter at Central Records

Why it's the wildcard
  • Nobody else can ship this, the world IS a real execution trace
