const MAX_ITERATIONS =
  Math.max(2, Math.min(3, n || 3));

for (let i = 1; i <= MAX_ITERATIONS; i++) {
  const run = await agent(diagnose);

Untrusted paths are data, not instructions

Coverage-report keys are attacker-influenced text that an agent will read. Stripping control characters does not make them safe to place in a sentence.

The mitigation is placement, not escaping: paths are fenced, labelled untrusted, and referenced positionally ("the FIRST file listed above"). This bounds the blast radius; it does not eliminate it, and the docstring now says so.

Intent picks who. Weight picks how expensive.

The router classified intent only and never considered task weight, so nothing selected a cheap model for cheap work.

Resolution is asymmetric: ANY heavy signal wins; Light requires ALL light signals. Under-powering a security review yields a confident wrong answer nobody catches; over-powering a rename only wastes money.
